Projects I maintain:

  • LevelSpace, an extension for NetLogo for multi-level agent-based modeling. It allows NetLogo models to open up and interact with other NetLogo models. This allows researchers to create a system of interacting models, each modeling particular aspects of their target system. This is the focus of my dissertation. Written in Scala.
  • NetLogo Python extension, which allows NetLogo to run Python code, allowing modelers to use sophisticated data analysis libraries like numpy, scipy, and TensorFlow in their NetLogo models. Written in Scala and Python.
  • NetLogo ControlFlow extension, which has allowed us to experiment with new control flow structures in NetLogo. Written in Scala.
  • NetLogo CSV extension, an simple extension for parsing and writing CSV files in NetLogo. This allows models to easily incorporate common sources of data, and record their own data as they run.
  • NetLogo NW extension, an extension for complex network analysis in NetLogo. It contains many tools
